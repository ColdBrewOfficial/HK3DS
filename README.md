# HK3DS
This repo hosts the Hollow Knight 3DS Demake source code and CIA releases.

# Build
The source code is not currently released. HOWEVER, once you would like to build, follow these instructions:

## Step 1
Download the source code

## Step 2

Download the Unity version for building for 3DS

https://drive.google.com/file/d/1XGogKpzP7WBkkE8cUhqJceFz8T2XrdKL/view

## Step 3
Once the Unity version is installed, use Unihacker to patch the Unity.exe file

https://archive.org/details/uni-hacker-v-4.5

(Select the Unity.exe, not the Unity installer)

## Step 4

Open the project in the unity 5.6 you installed earlier

## Step 5

Set the build target to 3DS

## Step 6

build for 3DS, but do not press "Build And Run," and make sure that none of the checkboxes are checked (no CIA, no Development build)

## Step 7

Once built, you will get a .cci file (which is just a .3ds file) you can now install it, or convert it to a .cia file using GM9
